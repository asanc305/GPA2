THE SEMESTER DASHBOARD

The Semester Dashboard provides a way for students to keep track of how they are doing in a certain course. For each course a user can create a set of "Assessment Types" which are basically a collection of grades that have the same weight. One possible example is a course where homework is worth 30% of the grade. The user can then create an Assessment Type called "Homework" with a weight of 30%. The user can insert what grades they got for assessments in that Assessment Type. The system will persistently store information about grades and automatically calculate total running average for the whole semester.
